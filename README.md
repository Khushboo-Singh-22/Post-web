# 📸 Post Web App (MERN Stack)

A full-stack web application where users can create, view, and delete posts with image uploads.

## 🌐 Live Demo
- 🔗 Frontend: https://post-web-jmgl.vercel.app  
- 🔗 Backend:  https://post-web-21mt.onrender.com

## 🚀 Features
- Create post with image & caption
- View all posts (feed)
- Delete post
- Responsive UI
- Image upload using ImageKit

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Image Upload: ImageKit

## 📂 Project Structure
Backend/
Frontend/

## ⚙️ Setup Instructions

### Backend
cd Backend
npm install
npm run server

### Frontend
cd Frontend
npm install
npm run dev
